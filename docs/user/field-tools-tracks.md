# Field Tools and Tracks

Ark includes utility screens for quick offline checks and track recording.

## Tools

- Compass
- Barometer and pressure trend
- Level
- Pedometer
- Light meter
- Coordinates and saved places
- Readiness checklist
- Chronometer
- Cached weather
- Cached news feeds
- Diagnostics

Sensor availability depends on the device and native build. Diagnostics shows what is actually available.

## Tracks

Tracks record movement samples, elapsed time, markers, stats, chart previews, GPX export, and map overlays.

Background and locked-screen recording still need real-device proof for battery use, location continuity, encrypted database behavior, and notification recovery.
