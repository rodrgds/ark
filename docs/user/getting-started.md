# Getting Started

Ark is built for preparation work you do before a trip, outage, storm, or low-connectivity period.

## First Run

1. Complete onboarding.
2. Choose whether vault passphrase protection is enabled now or later.
3. Grant location and notification permissions if you plan to use maps, route downloads, track recording, or download recovery alerts.
4. Pick starter packs conservatively. Large Wikipedia, model, and map downloads should wait until you are on stable Wi-Fi and power.
5. Open Settings and check the readiness rows for maps, downloads, AI, storage, and diagnostics.

## Prepare For Offline Use

- Download at least one offline map region for the area you expect to use.
- Open the Library and install guides or ZIM archives you need.
- Import key PDFs or local documents before leaving connectivity.
- Create or import notes you may need under stress.
- Refresh weather and RSS feeds before going offline.
- Export an encrypted backup if the device will be your only copy.

## What Ark Does Not Replace

Ark does not replace emergency services, medical professionals, local official instructions, verified field training, or current hazard reports. Treat it as an offline organizer and retrieval tool, not an authority.
