# Knowledge Library

The Library is the offline content hub for guides, PDFs, ZIM archives, models, RSS, and imported documents.

## Content Types

- Authored guides are bundled Markdown references for quick reading.
- PDF packs include first-aid, survival, and public reference manuals.
- ZIM archives bring Kiwix-style offline knowledge into Ark when the native reader is available.
- RSS feeds cache official updates for later reading.
- Imported documents can be searched and cited by local source search after extraction.
- Model packs are optional and large. Install them only when you need local AI answers.

## Downloads

Downloads show byte progress, verification progress, terminal states, and retry actions. Ark rejects empty downloads and tries to verify checksums when metadata exists.

## ZIM Archives

Android and iOS native ZIM bridges are wired, but large-archive validation still needs real-device proof. If the in-app reader is unavailable, Ark can hand the archive to another installed reader.
